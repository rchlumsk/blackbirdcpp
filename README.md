# Blackbirdcpp 

<img src="src/blackbirdlogo.png" align="right" style="width: 400px; " />

The code repository for the compiled program version of Blackbird, developed by Heron Hydrologic Ltd.

Additional resources and links are available on the [Heron Hydrologic website](https://heronhydrologic.ca/blackbird/).

If you would like to work with the active development of Blackbird, please do so on a branch and coordinate with the Blackbird development team.

Please contact us if you are developing or trying out the program!

## Windows Users

Pre-built executables are available for Windows users on the [Heron Hydrologic website](https://heronhydrologic.ca/blackbird/).

## Building Blackbird

For more advanced uers, the program can also be compiled from source on multiple platforms.
